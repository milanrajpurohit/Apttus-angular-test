# IndiaNIC Angular Project

## Prequisites (Development):

| Module | Version |
| --- | --- |
| Node | 8.9.4 |
| Npm | 5.6.0 |
| Angular-cli | 1.6.0 |
| Angular | 5.1.1 |


##### Take Clone of project
> git clone -b git_url  folder_name

##### Run Project

> npm install

> ng serve

##### Deployment
> npm run build

Deploy the dist/browser folder to your server.
